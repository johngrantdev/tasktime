# Tasktime Communication Service

## Description

This service is used for sending emails utilizing Handlebars and Nodemailer.

## Dependancies

NAT Messaging Server
Nodemailer - For sending 'magic link' login emails

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
